# 弹幕配置指南

## 🎯 弹幕服务器配置

现在你可以在 Swiftfin 应用中配置弹幕服务器地址了！

### 📍 配置位置

1. **打开 Swiftfin 应用**
2. **进入设置** → **视频播放器设置**
3. **找到弹幕设置部分**
4. **在"服务器配置"部分配置弹幕服务器**

### ⚙️ 配置选项

#### 弹幕服务器地址
- **字段名**: 弹幕服务器地址
- **默认值**: `http://192.168.50.112:8080/danmu/api/danmu`
- **说明**: 输入你的弹幕服务器完整地址

#### 弹幕源选择
- **Jellyfin**: 使用 Jellyfin 弹幕插件
- **DanDanPlay**: 使用 DanDanPlay API
- **自定义**: 使用自定义弹幕源

### 🔧 服务器地址示例

#### Jellyfin 弹幕插件
```
http://your-jellyfin-server:8096/danmu/api/danmu
```

#### DanDanPlay API
```
https://api.dandanplay.net/api/v2/comment
```

#### 自定义服务器
```
http://192.168.1.100:8080/danmu/api/danmu
https://your-domain.com/danmu/api/danmu
```

### 📝 配置步骤

1. **输入服务器地址**
   - 在"弹幕服务器地址"字段中输入完整的服务器地址
   - 确保地址格式正确（包含 http:// 或 https://）

2. **选择弹幕源**
   - 根据你的服务器类型选择对应的弹幕源
   - Jellyfin 插件选择"Jellyfin"
   - DanDanPlay API 选择"DanDanPlay"
   - 其他自定义服务器选择"自定义"

3. **保存设置**
   - 设置会自动保存
   - 重新播放视频时新设置会生效

### 🎮 使用弹幕

配置完成后：

1. **播放视频**
2. **点击弹幕按钮** (💬) 启用弹幕
3. **弹幕会自动加载并显示**

### 🔍 其他弹幕设置

在弹幕设置中，你还可以调整：

- **启用/禁用弹幕**: 总开关
- **不透明度**: 10% - 100%
- **字体大小**: 12 - 24
- **滚动速度**: 0.5x - 2.0x
- **最大显示数量**: 5 - 50 条
- **弹幕类型过滤**:
  - 显示顶部弹幕
  - 显示底部弹幕
  - 显示滚动弹幕

### ⚠️ 注意事项

1. **网络连接**: 确保设备能访问配置的弹幕服务器
2. **服务器兼容性**: 确保服务器支持预期的API格式
3. **地址格式**: 服务器地址必须包含协议（http:// 或 https://）
4. **端口号**: 如果服务器使用非标准端口，请包含端口号

### 🐛 故障排除

如果弹幕无法加载：

1. **检查服务器地址**: 确保地址正确且可访问
2. **检查网络连接**: 确保设备能连接到服务器
3. **查看日志**: 在 Xcode 控制台中查看错误信息
4. **测试服务器**: 在浏览器中测试服务器地址是否可访问

### 🎉 完成！

现在你的 Swiftfin 应用已经支持完整的弹幕配置功能了！
