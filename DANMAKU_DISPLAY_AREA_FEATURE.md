# 弹幕显示区域功能实现

## 🎯 功能概述

参考腾讯视频的弹幕显示效果，实现了弹幕显示区域控制功能，让弹幕只在屏幕的指定区域显示，而不是占满整个屏幕。

## ✨ 新增功能

### 1. 弹幕显示区域设置

- **显示区域大小**：可设置弹幕显示区域占屏幕的比例（20%-100%）
- **显示位置**：支持顶部、底部、全屏三种位置
- **轨道数量**：可设置弹幕轨道数量（2-8条）

### 2. 设置界面增强

在 **设置 → 视频播放器设置 → 弹幕设置** 中新增：

#### 基础设置
- ✅ 启用弹幕
- ✅ 不透明度 (10%-100%)
- ✅ 字体大小 (12-24)
- ✅ 滚动速度 (0.5x-2.0x)
- ✅ 最大显示数量 (5-50条)

#### 新增显示区域设置
- 🆕 **弹幕轨道数** (2-8条)
- 🆕 **显示区域大小** (20%-100%)
- 🆕 **显示位置** (顶部/底部/全屏)

#### 弹幕类型设置
- ✅ 显示顶部弹幕
- ✅ 显示底部弹幕
- ✅ 显示滚动弹幕

## 🔧 技术实现

### 1. 新增配置项

在 `SwiftfinDefaults.swift` 中添加：

```swift
static let danmakuDisplayArea: Key<Double> = UserKey("danmakuDisplayArea", default: 0.5)
static let danmakuAreaPosition: Key<String> = UserKey("danmakuAreaPosition", default: "top")
```

### 2. 渲染器增强

在 `DanmakuRenderer.swift` 中实现：

#### 显示区域计算
```swift
// 计算显示区域
displayAreaHeight = containerSize.height * displayAreaRatio

switch displayAreaPosition {
case "top":
    displayAreaOffsetY = 0
case "bottom":
    displayAreaOffsetY = containerSize.height - displayAreaHeight
case "full":
    displayAreaHeight = containerSize.height
    displayAreaOffsetY = 0
}
```

#### 轨道位置调整
```swift
// Y位置计算包含显示区域偏移
let yPosition = displayAreaOffsetY + CGFloat(trackIndex) * trackHeights[trackIndex] + (trackHeights[trackIndex] - textHeight) / 2
```

### 3. 设置界面更新

在 `DanmakuSection.swift` 中添加新的设置控件：

- **弹幕轨道数**：BasicStepper (2-8条)
- **显示区域大小**：BasicStepper (20%-100%)
- **显示位置**：SegmentedPicker (顶部/底部/全屏)

## 🎨 显示效果

### 默认设置（类似腾讯视频）
- **显示区域大小**：50%
- **显示位置**：顶部
- **轨道数量**：4条

这样弹幕只会在屏幕上半部分显示，不会遮挡重要的视频内容。

## 🚀 使用方法

### 1. 进入设置
1. 打开 Swiftfin 应用
2. 进入 **设置**
3. 选择 **视频播放器设置**
4. 找到 **弹幕设置** 部分

### 2. 配置显示区域
1. 调整 **显示区域大小** 滑块（推荐50%）
2. 选择 **显示位置**（推荐顶部）
3. 设置 **弹幕轨道数**（推荐4条）

### 3. 测试效果
1. 播放视频并启用弹幕
2. 观察弹幕是否只在设定区域显示
3. 根据需要调整设置

## 🔍 问题排查

### 1. 弹幕仍然全屏显示
**可能原因**：设置没有生效
**解决方案**：
- 检查显示位置是否设置为"全屏"
- 重新启动应用
- 查看控制台日志确认设置值

### 2. 弹幕显示区域太小
**可能原因**：显示区域大小设置过小
**解决方案**：
- 增加显示区域大小到50%以上
- 增加轨道数量到4-6条

### 3. 弹幕重叠
**可能原因**：轨道数量不足
**解决方案**：
- 增加轨道数量
- 减少最大显示数量
- 调整显示区域大小
